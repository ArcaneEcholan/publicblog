idea阻塞下载源码卡住整个idea界面了

自己去这个地址下载源码：  https://repo1.maven.org/maven2/

比如这个依赖，可以 net.java.dev.jna -> net/java/dev/jna

```
<groupId>net.java.dev.jna</groupId>
<artifactId>jna</artifactId>
<version>4.5.2</version>
```
