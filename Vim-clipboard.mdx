#### 常见寄存器：

| 寄存器名 | 用途说明 |
| --- | --- |
| `"` | 默认寄存器（所有 yank/delete 都覆写） |
| `0` | yank 的文本 |
| `-` | 行内剪切的文本（包括visual模式行内 delete） |
| `1`\-`9` | 行级剪切的文本(包括dd和visual模式跨行 delete），栈 |
| `a`\-`z` | 用户指定的寄存器（例如 `"ayy` 把当前行复制到 `a`） |
| `:` | 最近的命令历史 |
| `/` | 最近搜索的内容 |
| `.` | 上次插入的文本 |

关于 unnamed(*)以及unnamdplus(+)寄存器：https://vimhelp.org/options.txt.html#%27clipboard%27
