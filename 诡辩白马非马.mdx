# “白马非马”对话的背景

时代：战国时期（很多学者在争论各种问题的时代）。

人物：公孙龙（当时有名的辩论高手）和一位来访的“客人”。

地点：没有明确写在哪，但一般是公孙龙的住处或门客们聚会的场合。

目的：他们在讨论语言和逻辑问题，想证明“名字”和“东西”不是一回事。

# 大致内容

公孙龙的论证如下：

白马有两个特征：
* 有马的特征
* 白色的

而马只有一个特征：
* 有马的特征

# 为什么是诡辩

在日常语言中，“是”主要有二义：

**包含**: “是”用来指一语词所指的事物包含在一类事物中，例如月亮是卫星（的一种）、面包是食物（的一种）等。

**等同**: 表示两语词所指的事物等同，例如伦敦是英国的首都，三角形是有三条边的平面图形等。

据此，与“是”相反的“非”或“不是”，也主要有二义——不包含及不等同的意义。

他否定别人说“白马是马”，这明显是诡辩，因为这是一般人更常使用的“是”的意义，是指“白马”包含于“马”中，这只是与公孙龙自己使用“是”的意义不同，而不是这样使用“是”是错误的

# 附录

**诡辩**

诡辩是一个用于欺骗别人的混淆或不合逻辑的论点，也就是外表上、形式上好像是运用正确的推理手段，实际上违反逻辑规律，做出似是而非的推理。

诡辩术与辩证法有相似和不同之处：

* 相似处：辩证法，针对已经给出的论证，从普遍接受前提出发的，推理出与已给定论的矛盾。诡辩术也是如此。

* 不同之处：诡辩术在主观方面以欺骗为目的，客观上在论证中故意引入违背逻辑的谬论（Fallacy），做出有欺骗性的似是而非的推理和论证。 而辩证法主观方面没有以欺骗为目的，客观上尽力避免违背逻辑的谬论（Fallacy）。

**基本概念**

属（genus）：一个更大、更宽泛的分类，比如“动物”。

种（species）：属的下位分类，更具体，比如“马”。

**下定义的一个办法**

概念内涵：（定义） 概念的定义 = 种差（于其他种的差别） + 属（临近的属）。

目的是强调其他种的区别。
